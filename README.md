# Online Complaint Registration and Management System

A full-stack MERN web application designed to simplify complaint registration, tracking, assignment, and resolution.

## Features
- User registration and login
- Complaint submission and tracking
- Role-based dashboards for User, Agent, and Admin
- Agent assignment workflow
- Feedback collection
- Admin analytics dashboard
- Responsive React UI
- Express.js REST API structure
- MongoDB/Mongoose models

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- CSS

## Run Project

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm run dev
```
